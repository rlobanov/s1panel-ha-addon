# s1panel-ha-addon
HomeAssistant ddon for controlling the display of miniPCs such as AceMagic S1 or Firebat S1.

Based on the repository https://github.com/adriebaselmans/AceMagic-S1-LED-TFT-Linux.

Install:
- In HA: Settings > Add-ons > Add-ons Store.
- In the menu (three dots): Repositories > Add URL https://github.com/rlobanov/s1panel-ha-addon > Add.
- Update the store. Install "AceMagic S1 Panel."
- After installation, launch it. The GUI is available in the sidebar (mdi:monitor-dashboard icon).
